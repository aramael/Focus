Focus Help

How do I manually disable Focus?

Focus takes extreme care to only block your Internet when you wantâ€”but if it's broken and you need to manually disable it, go to System Preferences -> Network -> Advanced -> Proxies -> Uncheck Automatic Proxy Configuration.

How can I start Focus from the command line?

There are two very simple scripts to do this, focus and unfocus. Look at the source to see how easy it is to hook into Focus and control it from your own scripts.

Does Focus read any of my traffic?

No. Focus does setup a local proxy server to intercept requests, but everything is self contained on your computer.

Does Focus work on HTTPS?

Focus will block HTTPS sites, but can't display a custom message.

What if I have an existing PAC file?

Focus currently won't work if you use an existing PAC file, but this will be fixed in a future update.

Is it possible to run Focus in my dock rather than menubar?

Currently no but this is a planned feature.

Do you have a grayscale icon? I don't like colorful icons in my menubar.

Yes! This has been added as a configurable option.

Focus doesn't work in Firefox

Firefox has a way to override the system-wide proxy which can sometimes cause problems. To fix this go to Firefox, Preferences, Advanced, Network, Settings and change the selected option to "Use system proxy setting"
